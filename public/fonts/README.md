# Font Files Required

To complete the marketing design system integration, you need to copy the SuisseIntl font files from the marketing app:

1. Create the directory structure:
   ```
   public/fonts/SuisseIntl/
   ```

2. Copy these files from `/Users/nit/Developer/firecrawl/firecrawl-marketing/public/fonts/SuisseIntl/`:
   - 400.woff2
   - 450.woff2
   - 500.woff2
   - 600.woff2
   - 700.woff2

3. Place them in `/Users/nit/Developer/firecrawl/firecrawl-web/public/fonts/SuisseIntl/`

These fonts are essential for matching the marketing site's typography.
